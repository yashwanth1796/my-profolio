## Getting Started

First, install dependencies:
```bash
npm install
# or
yarn
```
Then run the development server:
```bash
npm run dev
# or
yarn dev
```

Run Sanity (localhost:3333)


#### **`!!! Make sure to have installed Sanity globally`**
Then:
```bash
cd sanity && sanity start
```
It runs local server at localhost:3333
